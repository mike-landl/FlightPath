Detailed Description

Formatter specialization for FlightPath::Position.

This specialization allows FlightPath::Position to be used with std::format. It converts the longitude and latitude values from radians to degrees, and formats the position in a human-readable string that includes:

  • Longitude in degrees (6 decimal places)
  • Latitude in degrees (6 decimal places)
  • Altitude in meters (2 decimal places)
